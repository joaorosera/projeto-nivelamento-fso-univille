## 📖 Sobre o Projeto

O **Descarte Consciente de Pilhas** é uma plataforma web informativa e utilitária que visa proteger o meio ambiente e a saúde pública. O descarte incorreto de pilhas libera metais pesados (como cádmio, chumbo e mercúrio) que contaminam o solo, a água e trazem graves riscos à saúde humana.

Através deste site, os usuários podem:
- Entender os perigos do descarte incorreto para a água e saúde.
- Encontrar **Pontos de Coleta** adequados.
- Fazer uma **Solicitação de Coleta** para volumes maiores.
- Realizar uma **Denúncia** de descartes irregulares na natureza.

## ✨ Funcionalidades

- 🏠 **Início:** Informações cruciais sobre o impacto ambiental das pilhas.
- 📍 **Pontos de Coleta:** Mapeamento de locais adequados para o descarte seguro.
- 🚛 **Solicitação de Coleta:** Formulário dedicado para solicitar o recolhimento de materiais.
- 🚨 **Denúncia:** Canal para relatar descartes incorretos e crimes ambientais.
- ✅ **Sucesso:** Tela de feedback amigável para o usuário após interações nos formulários.
